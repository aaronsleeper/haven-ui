Usage Notes

  • Layout classes: .issues-layout wraps a .issues-layout-main + .issues-layout-aside pair.
  • Responsive: Below 1024px, the aside becomes a fixed right-side overlay with backdrop. Toggle it with the [data-issues-toggle] button.
  • Sticky: The .issues-sidebar class makes the card sticky on scroll (desktop).
  • Severity: Use text-error-500 for critical, text-warning-500 for warnings, text-info-500 for informational.
  • Resolved state: Add .issue-item-resolved to resolved items. Use .issues-sidebar-resolved for the all-clear message.
