Haven Pattern Library

Component reference for Cena Health applications

The Haven Pattern Library is the single source of truth for UI components used across Cena Health applications. Every component shown here maps to a semantic CSS class defined in components.css, ready to drop into any page without utility-class soup.

Navigate by category using the sidebar on the left. Every page opens with a Component info panel naming the class list, usage guidance, and Preline dependency; the live demo follows below.

Agents: start at pattern-library/COMPONENT-INDEX.md for the full ground-truth list, then open the component file and copy the marked markup block. Designers and engineers: read DESIGN.md at the repo root for the canonical brand and token spec; use the sidebar to jump to any component.
