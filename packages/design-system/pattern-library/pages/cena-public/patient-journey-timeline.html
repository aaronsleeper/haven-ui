  • No emphasis modifier on any cell (deliberate breadth-not-climax call per blueprint document register)
  • Generic-AMC framing throughout — every cell uses "the partner" / "the partner's clinician" phrasing; the strict-superset UConn-named upgrade lands when Vanessa clearance does
  • HDG row 3 — no vendor stack names anywhere in the rendering
  • HDG row 4 — clinical-staff cells use partner-clinician phrasing per the accountability model

Visual-language calls deferred to Haven Visual Designer

Four visual-language calls are deliberately deferred to a Haven Visual Designer expert pass. v0.1 ships with the existing canon set (no inline modifier candidates added) so the deferred calls don't get foreclosed by a freelance render decision.

  • Lane background tints. Each lane could carry a subtle background tint (e.g., patient lane white, clinical lane sand-50, substrate lane sand-100 underneath the substrate-fill boxes, provenance lane a darker sand-100 to anchor it visually as the audit-trail substrate ground). v0.1 uses no lane background tints; all lanes share the same diagram-frame ground. HVD call.
  • Substrate-uniformity register. All 9 substrate cells use the existing .diagram-box--substrate variant uniformly. HVD should evaluate whether the uniformity reads as "by the system, automatically" (intended) or as visually monotonous (unintended). Promoting a register-differentiated set of substrate variants is HVD's call if the uniformity reads as the wrong density.
  • Provenance-lane treatment. v0.1 uses the default .diagram-box for all 9 provenance cells. The Phase 3a audit §3 brand-fidelity considerations noted that .diagram-box--milestone-done (green-50 fill) reads as "outcome achieved" which collides with the artifact-voice constraint. Promoting a new .diagram-box--provenance variant (e.g., a quieter sand register with a hairline accent) for register differentiation is HVD's call.
  • Stage-column-header treatment. v0.1 stages each column with tiny-caps Source Code Pro headers matching the .diagram-lane-label register. Background tint, separator weight, and vertical-rhythm hooks are deferred to HVD.
