Diagram Frame

Outer SVG container with built-in accessibility handles and viewport-scaling. The frame standardizes role="img" + aria-label, optionally pointing to a sibling diagram-long-desc for screen-reader-friendly prose descriptions of complex diagrams.

With diagram-long-desc (sr-only canonical 4-section structure)

For complex diagrams, pair the frame with a sibling diagram-long-desc block via aria-describedby. Canonical structure — purpose → elements → relationships → optional notes. Purpose-line voice mirrors Cena brand voice — observational and specific, naming actors and what they do. Avoid generic framings ("a diagram showing system components"). Authoring this block well makes the diagram comprehensible to screen readers AND becomes the long-form documentation other teams cite.
