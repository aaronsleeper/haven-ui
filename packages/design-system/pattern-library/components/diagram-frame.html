Diagram Frame

Outer SVG container with built-in accessibility handles and viewport-scaling. The frame standardizes role="img" + aria-label, optionally pointing to a sibling diagram-long-desc for screen-reader-friendly prose descriptions of complex diagrams.

With diagram-long-desc (sr-only canonical structure)

For complex diagrams, pair the frame with a sibling diagram-long-desc block via aria-describedby. The long-desc structure is canonical: purpose → elements → relationships → optional notes. Purpose-line voice mirrors Cena brand voice — observational and specific, not generic.
