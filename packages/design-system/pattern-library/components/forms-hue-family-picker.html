Hue Family Picker

Single slider along a Haven hue family's OKLCH line. Drag the thumb to pick any continuous point between the family's min anchor (stop-950, darkest) on the left and max anchor (stop-50, lightest) on the right, with root (stop-500) at the midpoint. Canonical stops snap at every 10% interval; intermediate values render as derived stop numbers (e.g. value 35 → teal-650).

Compact variant — for stacked use

The dense variant intended for theming editors that stack 8+ pickers as theme "anchors." Add .is-compact to the root and include a chevron toggle next to the HEX readout to reveal OKLCH/RGB/HSL in an expand drawer. Each row collapses to ~56–64px so a stack of a dozen reads as a navigable list rather than a wall.

Tick labels recede to hover at this density — hover any tick to confirm its stop number; click jumps to that stop.
