Chat Button Row

Inline button affordance row for the patient-app chat thread. Single + dual + has-helper variants. Tier comes from the btn-* child.

Single — secondary (advancement)

Dual — secondary × 2 (irreversible-feeling)

Asymmetric — advancement + commit pair (Skip + Save, gap-doc C-4)

has-helper — primary commit, helper-gate (disabled, aria-describedby)

Add at least one item to continue.

has-helper — primary commit, helper-consequence (enabled)

Sends to your dietitian for review.
