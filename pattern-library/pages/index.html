Haven Pattern Library

Component reference for Cena Health applications

The Haven Pattern Library is the single source of truth for UI components used across Cena Health applications. Every component shown here maps to a semantic CSS class defined in components.css, ready to drop into any page without utility-class soup.

Agents: start with pattern-library/index.md to locate the right component file, then copy the marked markup block. Developers: browse the sidebar categories to find live examples with usage notes and class references.
