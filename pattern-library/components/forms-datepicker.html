Date Picker

Calendar-based date selection powered by Preline HSDatepicker. Renders as a popup from an input or inline.

Basic Date Picker

Click the input to open the calendar popup. Select a date to populate the field.

Preset Date

A datepicker initialized with a pre-selected date.

Date Range

Select a start and end date. Uses selectionDatesMode: "multiple-ranged".

Min / Max Date Constraints

Dates outside the allowed range are disabled. This example limits selection to March 2026.

Multiple Dates

Select multiple individual dates. Uses selectionDatesMode: "multiple".

Disabled

A non-interactive datepicker. The input and icon are dimmed.

Inline Calendar

An always-visible calendar rendered directly in the page. Uses inputMode: false.

Custom Date Format

Displays selected date with slash separators instead of dots.
