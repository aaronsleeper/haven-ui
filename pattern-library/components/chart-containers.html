Chart Containers

Sizing classes for Chart.js canvases. All extend .chart-canvas-wrapper as the base wrapper.

.chart-canvas-wrapper

Base wrapper class — all chart sizing classes extend this. Sets relative positioning, full width, and overflow hidden.

chart-canvas-wrapper (base — no fixed height)

.chart-sparkline

Compact 40px-tall sparkline. No axes, no labels, no legend.

.chart-line

Standard line chart at 240px height. Axes, grid lines, and tooltips visible.

.chart-bar

Bar chart at 200px height.
