Time Picker

Scroll-column time selector built on Preline hs-dropdown. Select hours, minutes, and optionally AM/PM.

Browser Default

Native HTML time input. Use as a lightweight fallback when the custom picker is not needed.

12-Hour Format

Hours 01–12 with AM/PM column. Click the clock icon to open the selector.

24-Hour Format

Hours 00–23 with no AM/PM column. Common for clinical and kitchen scheduling.

Preset Time

A time picker initialized with a pre-selected value. Radios are pre-checked and the input shows the time.

Disabled

A non-interactive time picker. The input and icon button are dimmed.

Inside a Field Row

Time picker composed with the field-row pattern including label and help text.
