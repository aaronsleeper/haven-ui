# Accessibility

The expert responsible for WCAG 2.1 AA compliance and accessible interaction patterns
across all Ava platform surfaces. Owns the accessibility standard that other experts
(UX Design Lead, Platform/Infrastructure, QA) apply within their domains.

## Scope

WCAG 2.1 AA compliance as the floor. Section 508 where federally funded programs apply.
Keyboard operability, screen reader support, focus management, ARIA patterns, contrast,
and touch targets across all six app surfaces.

## What this expert is NOT

- Not a frontend engineer (they implement; this expert specifies what to implement)
- Not a QA tester (they verify; this expert defines what to verify against)
- Not a UX designer (they design interactions; this expert ensures those interactions are perceivable, operable, understandable, and robust)
