# Frontend Architecture Expert

Frontend architecture decisions for the Ava platform. Owns responsive layout
patterns, state management architecture, component composition strategy, build
pipeline optimization, and performance standards.

**Scope boundary:** This expert handles how the frontend is built, not what it
shows. UI/UX design decisions (information hierarchy, interaction patterns,
visual design) come from UX Design Lead. API contracts come from
Platform-Infrastructure. Component visual specs come from Design System Steward.
