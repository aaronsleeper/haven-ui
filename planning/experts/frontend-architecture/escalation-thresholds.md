# Escalation Thresholds

| Action | Tier | Condition |
|---|---|---|
| Add responsive classes to existing components | Autonomous | CSS-only changes following established breakpoint patterns |
| Create new @ava/ui component | Notify | Inform Aaron — new shared component affects all apps |
| Change ThreePanelLayout structure | Gate | Layout is the core UX pattern — requires Aaron's approval |
| Adopt new state management library | Gate | Architectural shift — requires discussion |
| Change build pipeline (Turborepo config, Vite config) | Notify | Build changes can break DX for all developers |
| Mobile navigation pattern (bottom nav vs sidebar) | Gate | UX decision with architecture implications — needs UX Design Lead |
