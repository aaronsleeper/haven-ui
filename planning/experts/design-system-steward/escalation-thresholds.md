# Escalation Thresholds

| Action | Tier | Condition |
|---|---|---|
| Flag duplicate pattern for extraction | Autonomous | Meets extraction criteria in judgment framework |
| Propose canonical API for extraction | Notify | Draft and present for review |
| Add new semantic token | Notify | Follows naming convention, aliases existing palette value |
| Rename existing token | Gate | Requires codemod plan and approval before execution |
| Remove component from @ava/ui exports | Gate | Requires consumer audit and approval |
| Change extraction criteria thresholds | Gate | Modifies judgment framework -- requires human approval |
